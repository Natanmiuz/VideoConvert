# Convertidor de Video

**Convertidor de Video** es una aplicación de escritorio de **código abierto** desarrollada en Python, que permite convertir archivos de video entre los formatos más populares (MP4, AVI, MOV, MKV) usando una interfaz gráfica intuitiva y el potente motor FFmpeg.

## Características

- Interfaz gráfica amigable y moderna.
- Soporte para múltiples formatos: MP4, AVI, MOV, MKV.
- Conversión rápida y eficiente con FFmpeg.
- Compatible con Windows.

## Instalación y Uso

1. Clona el repositorio:
   ```sh
   git clone https://github.com/Natanmiuz/VideoConvert.git
   cd VideoConvert
   ```
2. Instala las dependencias:
   ```sh
   pip install -r requirements.txt
   ```
3. Asegúrate de tener FFmpeg instalado y en tu PATH.
4. Ejecuta la aplicación:
   ```sh
   python main.py
   ```

## Estructura del proyecto

- `main.py`: Punto de entrada.
- `core/core.py`: Lógica de conversión.
- `gui/ui.py`: Interfaz gráfica.
- `requirements.txt`: Dependencias.
